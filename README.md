# 執行 package.json 指令

ex: npm run buildgit

# 更版部屬步驟

(1) npm run buildgit
(2) 打開 dist 資料夾將 browser 內的檔案剪下貼在 dist 下
(3) git add .
(4) git commit -m "更新內容說明"
(5) git push origin master
(6) 等待更新至 git 更新
